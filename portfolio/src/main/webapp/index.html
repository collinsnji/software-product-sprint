Projects

  • Snorkel for Data Labeling

    This project attempts to use Snorkel to create weak labels for large amounts of training data at Georgetown CSET using weak supervision. The data is primarily paper abstracts, whose relevance to various WIPO Application Areas we explore. This was my summer project at my last internship where I used weak supervision to label about 4GB worth of paper abstracts by writing weak supervision models based on Snorkel

  • Free Write (Alfred Prufrock Editior)

    This project uses Google's NLP APIs to create a mind map of someone's thought process, and builds a dynamic mind map that shows both sentiment of thoughts and relationship to other parts of text as the user writes in an editor. This project was inspired by Alfred Prufrock's works in literature. This project uses Node.js for the backend, D3.js for the mind map visualization and JS for the front end.
